# iyf-s10-week-06-kimbelywamboi8@gmail.com🌤 Weather Dashboard
This dashboard fetches live weather data for any city and displays:

Temperature, weather conditions, humidity, wind speed
Dynamic backgrounds based on current weather
Weather icons from OpenWeatherMap
Responsive layout with smooth animations

The project focuses on handling real-world asynchronous data, error handling, and creating a responsive, interactive user experience.

✅ Features
🔍 Search for any city
🌡 Display temperature, condition, humidity, wind speed
🌈 Dynamic backgrounds according to weather conditions
🌍 Weather icons from OpenWeatherMap
📱 Fully responsive and animated interface
⏳ Loading states for better UX
🚫 Error handling for invalid city names or network issues
⚡ Async/await used for API calls
Optional: Save favorite cities (using local storage)
🛠️ Technologies Used
HTML5 – Structure
CSS3 – Styling and responsive layout
JavaScript (Vanilla) – Async operations and DOM manipulation
OpenWeatherMap API – Live weather data
Fetch API + async/await – Handling API requests
📁 Project Structure
weather-dashboard/
│
├── index.html      # Dashboard structure
├── style.css       # Styling & animations
└── script.js       # Async JS logic, API calls, error handling


##▶️ How to Run the Project
Clone or download the repository
Open the project folder in VS Code
Open index.html in a browser (or use Live Server for best experience)
Enter a city name in the search box and click Get Weather



##⚡ Async & API Highlights
Callbacks: Sequential operations (optional exercises)
Promises: Handling multiple asynchronous tasks
Async/Await: Clean, readable API calls
Error Handling: Alerts for invalid cities or network errors
Loading States: Show visual feedback while fetching data
