# SkyCast Weather App

SkyCast is a lightweight weather app built with HTML, CSS, and vanilla JavaScript. It allows users to search for a city, view current weather conditions, and check a 5-day forecast using live Open-Meteo data.

## Preview

Main weather app page:

- `weather.html`

Supporting files:

- `styles.css`
- `script.js`

Portfolio homepage:

- `index.html`

## Features

- Search weather by city name
- Live current conditions
- Feels-like temperature, wind speed, humidity, and daily range
- 5-day forecast
- Responsive layout for mobile, tablet, and desktop
- No API key required

## Tech stack

- HTML5
- CSS3
- JavaScript
- Open-Meteo Geocoding API
- Open-Meteo Forecast API
